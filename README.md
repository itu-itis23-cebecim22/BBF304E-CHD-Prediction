## Project Summary

Binary classification system to predict 10-year coronary heart disease risk using the Framingham Heart Study dataset. Three classical machine learning models are compared: Logistic Regression, Decision Tree, and Random Forest.

**Best result:** Logistic Regression — ROC-AUC: 0.6990

---

## Dataset

Framingham Heart Study (publicly available subset)  
**Link:** https://www.kaggle.com/datasets/dileep070/heart-disease-prediction-using-logistic-regression

4238 records, 15 features, 1 binary target (TenYearCHD)
